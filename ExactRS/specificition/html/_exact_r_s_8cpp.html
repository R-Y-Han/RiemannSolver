◆ identifyNonlinearWave()

wavetype identifyNonlinearWave ( double  p)

Identify the type of two nonlinear waves.

  1. If p* > pl, the left wave is a shock;
  2. If p* < pl, the left wave is a rarefaction;
  3. If p* > pr, the right wave is a shock;
  4. If p* < pr, the right wave is a rarefaction.
参数
[double]p : pressure input, p = pl or pr
返回
wavetype :
  1. When p = pl, return the type of left wave;
  2. When p = pr, return the type of right wave.
